# Depot Power Planner

Depot Power Planner is a simple planning tool for estimating electric bus depot charging power needs.

The tool allows users to explore how fleet size, battery size, charging windows, depot baseload, priority charging and charging strategy affect required depot power.

## Features

- Estimate required depot power for electric bus charging
- Compare fixed charging sessions and flexible smart charging
- Include priority buses that must charge first
- Include constant depot baseload
- Check whether a depot power limit is sufficient
- Export results as CSV and PDF

## Intended users

The tool is intended primarily for public transport authorities, public transport operators, researchers and planners working with electric bus depot charging.

## Disclaimer

This tool provides first-order planning estimates based on simplified charging assumptions. It is intended to support early-stage planning and should not replace detailed depot design or operational simulation.
